# Кастомная "новая вкладка" от Zavar'ыча

Уээээ... Замутил свою "новую вкладку" для хрома, пользуйтесь.<br>
Базовый минимум там впринципе есть, если чего не хватает - не проблема добавить.

## Как использовать

> 1. Скачиваешь [последний резиз](https://github.com/zavarich/CustomChromeTab/releases).
> 2. Распаковываешь в любое место на свем ПК.
> 3. Заходишь в "Расширения" ``` chrome://extensions/ ```
> 4. В правом верхнем углу будет тумблер "Режим разработчика", включи его.<br>
> <img src="./readme/4step.png" /><br>
> 5. Добавляешь расширение.<br>
> <img src="./readme/5step.png" />

> [!WARNING]
> Необходимо загрузить именно папку ``` extention ```, так как в ней находится ``` manifest.json ```, необходимый для работы расширения.

> 6. Готово, теперь все последущие "новые вкладки" будут заменены на новую.

> [!WARNING]
> "Новые вкладки" открытые до установки расширения останутся без изменений.
